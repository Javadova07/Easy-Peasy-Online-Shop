# Easy-Peasy-Online-Shop
Demo E-Commerce Website built with Laravel .
Modern e-commerce website developed using Laravel and Bootstrap.

## Features

- User authentication
- Product listing
- Product details page
- Multi image upload
- Shopping cart
- Checkout system
- Order management
- Admin dashboard

## Database

- Products table
- Orders table
- Order items table
- Product images table


## Installation

bash
git clone https://github.com/Javadova07/Easy-Peasy-Online-Shop.git

composer install

php artisan migrate

php artisan serve
